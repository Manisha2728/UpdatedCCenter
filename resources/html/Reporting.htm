Defines the caption used for the Patient Identifier field in the report header. For example, MRN or PHIN.
This applies to:
- Clinical Viewer: RTF, TIF and Clinical Summary selected items reports.
- Collaborate: All reports.
- EHR Agent: All reports.
- Patient List: All reports.
- Patient View: All reports.
Default: MRN

Defines the Font Size to use in the report.
This applies to:
- Collaborate: TXT reports.
- EHR Agent: TXT reports.
- Patient List: TXT reports.
- Patient View: TXT reports.
Default: 9

Defines the Font Type in the header and footer of the report.
This applies to:
- Clinical Viewer: RTF, TIF and Clinical Summary selected items reports.
- Collaborate: All reports.
- EHR Agent: All reports.
- Patient List: All reports.
- Patient View: All reports.
Default: Arial

Defines the Date and Time format of the Report footer.
This applies to:
- Clinical Viewer: RTF, TIF and Clinical Summary selected items reports.
- Collaborate: All reports.
- EHR Agent: All reports.
- Patient List: All reports.
- Patient View: All reports.
Default: General date/long time

Determines whether the Clinical Viewer, Patient View, Collaborate and EHR Agent reports use the special disclaimer template (Word file) which displays the confidentiality disclaimer.
True: The system uses the special Disclaimer templates for reports.
False: The system uses the regular OOB templates for the reports (default).
Default: False

Determines whether to remove internal links in RTF documents, because these internal links occasionally produce error messages in the document upon conversion to PDF.
True: Removes RTF internal links. The links will still exist in the PDF file as text.
False: Does not remove RTF internal links.
Default: False

Determines how the Lab Results and Lab Results History clinical views and reports (in the Clinical Viewer, Collaborate, Patient List and EHR Agent applications) display the types of Microbiology Test results.
Default: MIC: Displays all types of test results in a MIC captioned column.

Determines whether to show the Lab Report Remark and Result text in fixed-width font.
True: Displays text in fixed-width font.
False: Does not display text in fixed-width font.
Default: True
