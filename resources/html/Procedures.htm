Time Range Filter Services to restart

  • IIS(Clinical Viewer)
  • IIS(CVA)

The textbox determines the time Range used to filter the patient's Procedures list when using the Show option.
The dropdown menu defines the time Unit used to filter the patient's Procedures list. For example, if the Unit is "month" and the Range is 10, the patient's Procedures over the past 10 months are displayed.
This configuration affects the Clinical Viewer and EHR Agent.
Default: 5 Years.

Code System Display Services to restart

  • IIS(Clinical Viewer)

Determines how to display the CodeSystem name.
This configuration affects the Clinical Viewer page and the Clinical Summary Selected Items report.
Default: Display the CodeSystem Name only as a tooltip when the user hovers the cursor over the Code column
