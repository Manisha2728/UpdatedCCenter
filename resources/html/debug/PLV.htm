Set the Clinical Viewer PLV Front End Settings

Applies to: Clinical Viewer

  • History

Time Range Filter Services to restart

  • IIS(Clinical Viewer)
  • IIS(CVA)

The textbox determines the time Range used to filter the patient's PLV list when using the Show option.
The dropdown menu defines the time Unit used to filter the patient's PLV list. For example, if the Unit is "month" and the Range is 10, the patient's PLV over the past 10 months are displayed.
Default: 1 Month.
