Time Range Filter Services to restart

  • IIS(Clinical Viewer)
  • IIS(CVA)

The textbox determines the time Range used to filter the patient's Allergies list when using the Show option.
The dropdown menu defines the time Unit used to filter the patient's Allergies list. For example, if the Unit is "month" and the Range is 10, the patient's Allergies over the past 10 months are displayed.
This configuration affects the Clinical Viewer and EHR Agent.
Default: 0: All. Displays all data (no filtering).
