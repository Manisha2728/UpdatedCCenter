Partitioning Details

This configuration DELETES old records from ArchMessage tables and moves them to DILMessagesArchive_Expired db as part of the database size reduction process. These messages will no longer be available in the DAT and the Replay tool.
Default: 11000 days (about 30 years). It is recommended NOT to change the default value unless database size reduction is requested.
