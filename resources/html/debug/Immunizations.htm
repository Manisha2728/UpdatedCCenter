Time Range Filter Services to restart

  • IIS(Clinical Viewer)
  • IIS(CVA)

The textbox determines the time Range used to filter the patient's Immunizations list when using the Show option.
The dropdown menu defines the time Unit used to filter the patient's Immunizations list. For example, if the Unit is "month" and the Range is 10, the patient's Immunizations over the past 10 months are displayed.
This configuration affects the Clinical Viewer and EHR Agent.
Default: 5 Years.

Determines whether the "Show Cancelled" checkbox is displayed in Clinical Viewer.
If False: Show Cancelled checkbox is not displayed, and cancelled items are not displayed in the Clinical Viewer.
If True: Show Cancelled checkbox is displayed. Its default behavior is based on the "Show Cancelled is selected" configuration.
Default: True
