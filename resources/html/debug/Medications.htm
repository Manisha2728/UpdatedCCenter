Time Range Filter Services to restart

  • IIS(Clinical Viewer)
  • IIS(CVA)

The textbox determines the time Range used to filter the patient's Medications list when using the Show option.
The dropdown menu defines the time Unit used to filter the patient's Medications list. For example, if the Unit is "month" and the Range is 10, the patient's Medications over the past 10 months are displayed.
This configuration affects the Clinical Viewer and EHR Agent.
Default: 1 Year.

Display Options Services to restart

  • IIS(Clinical Viewer)
  • IIS(CVA)

Determines whether the "Show Cancelled" checkbox is displayed in Clinical Viewer.
If False: Show Cancelled checkbox is not displayed, and cancelled items are not displayed in the Clinical Viewer.
If True: Show Cancelled checkbox is displayed. Its default behavior is based on the "Show Cancelled is selected" configuration.
Default: True


Determines whether the Show Cancelled checkbox (if displayed) is selected.
Default: False

Determines whether by default the Medications view filters out all inactive medications when the user first launches the Medications clinical view.
If True, the view only displays medications whose status is active.
If False, the view displays all medications.
Default: False

Determines whether to enable grouping of the Medications list. If True, the Grouping option is displayed. If False, no Grouping option is displayed.
Default: True

Determines whether, by default, Medications are grouped.
Default: True
