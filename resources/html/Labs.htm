Time Range Filter Services to restart

  • IIS(Clinical Viewer)
  • IIS(CVA)

The textbox determines the time Range used to filter the patient's Laboratory list when using the Show option.
The dropdown menu defines the time Unit used to filter the patient's Laboratory list. For example, if the Unit is "month" and the Range is 10, the patient's Laboratory over the past 10 months are displayed.
This configuration affects the Clinical Viewer and EHR Agent.
Default: 9 Months.

Lab Report Display Options Services to restart

  • IIS(Clinical Viewer)
The Lab Report column configuration is used to define how the Remarks column and the Results column of the Lab Report are displayed. Three basic display options are available, depending on the size of the text to be displayed. The display option is selected based on a series of verifications performed by the system based on the configured values of the parameters below.
The following options are available:
  • Regular Column Display: In this scenario, the text is not too large and can easily fit in the grid column area. To verify that the text can be displayed in the regular grid column, the system checks the text size according to the values configured in the following parameters: Result column configurations and Remarks column configurations.
  • Long Row Display: If the text size is too large and cannot be displayed in the regular grid column, the system provides an additional option, where the text is displayed in a long row provided either for the Result or the Remarks text (added immediately below the grid column) if either one of them is too large for the regular display. To verify that the text can be displayed in the long row, the system checks the text size according to the values configured in the following parameters.
  • End of Report Display: If the text size is too large for the Long Row (either in the Result or the Remarks column, or for both), the system provides an additional option, where the text is displayed at the end of the report regardless of the text size or the number of rows. Additional pages are provided to include the entire text.

Defines the maximum number of rows in the regular grid column.
This parameter is used to determine if the text can be displayed in the regular grid column.
Default: 20

Defines the total maximum number of characters in the Remarks column of the Lab Results Report.
This parameter is used to determine if the text can be displayed in the regular grid Remarks column.
Default: 600

Defines the maximum number of rows in a Long Row cell.
This parameter is used to determine if the text can be displayed in the Long Row.
Default: 33
